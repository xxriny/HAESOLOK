# 해소록 (Haesolog)

선생님의 하루를 기록하고, 마음의 무게를 덜어드릴게요.
**해소록**은 교육 공공데이터와 AI를 활용해 교사의 민원 대응과 번아웃 조기 감지를 돕는 교사 보호 서비스입니다.

---

## 🚀 실행 가이드 (Quick Start)

### 1. 사전 준비
- Node.js 18.x 이상 버전이 설치되어 있어야 합니다.

### 2. 프로젝트 설치
```bash
npm install
```

### 3. 환경 변수 설정 (.env)
프로젝트 루트 디렉토리에 `.env` 또는 `.env.local` 파일을 생성하고 다음 설정을 입력합니다. 
(현재 `.env.local.example` 파일이 템플릿으로 제공되고 있습니다.)

```bash
# LLM 제공자 선택: gemini | openai | anthropic | mock
LLM_PROVIDER=gemini

# 선택한 제공자의 API Key 입력 (예: Gemini 사용 시)
GEMINI_API_KEY=your_actual_api_key_here
GEMINI_MODEL=gemini-1.5-flash
```
*참고: `LLM_PROVIDER=mock`으로 설정하면 API 키 없이 시연용 데이터로 동작합니다.*

### 4. 개발 서버 실행
```bash
npm run dev
```
서버가 가동되면 브라우저에서 `http://localhost:3000`으로 접속하세요.

## 💡 AI 및 공공데이터 활용 원리

### 1. AI 답변의 근거
해소록의 AI(민원 대응 가이드)는 단순히 입력된 텍스트만 분석하지 않습니다. 다음 세 가지 요소를 결합하여 답변을 생성합니다.
- **교사 전문 프롬프트**: 교권 보호 관점에서 작성된 시스템 지침.
- **선생님의 기록**: 민원인 유형, 장소, 핵심 내용 등.
- **공공데이터 맥락**: 학사일정, 학교 규모, 관련 정책 키워드 등.

### 2. 공공데이터 연동 방식 (현재 MVP 기준)
현재 버전은 기술적 구조를 증명하기 위한 MVP 단계로, 공공데이터는 다음과 같이 연동되어 있습니다.
- **방식**: 실제 API 라이브 호출 대신, 민원 기록 시 시스템이 자동으로 **'가상 공공데이터 맥락(Mock Data)'**을 생성하여 AI에게 전달합니다.
- **대상 데이터**:
    - **나이스(NEIS)**: 학사일정 (예: 학부모 상담 주간 등)
    - **학교알리미**: 학교 규모 (예: 중대형 학교 현황 등)
    - **교육데이터맵**: 정책 및 법령 키워드 (예: 생활지도, 교권보호 등)

### 3. 향후 라이브 연동 계획
추후 각 기관의 Open API 승인을 통해, 온보딩 시 선택한 학교 식별자를 기반으로 **실제 오늘의 학사일정 및 최신 공공데이터**를 실시간으로 AI 프롬프트에 주입할 예정입니다.

## 🛠 주요 기능
- **온도 체크인**: 일일 정서 상태 기록 및 시각화.
- **민원 기록함**: 사실 중심의 기록 보존 및 AI 대응 가이드 생성.
- **주간 리포트**: 7일간의 온도와 민원을 분석한 정서 케어 인사이트.
- **공공데이터 안내**: 서비스에 활용된 데이터 출처 및 활용 방식 투명성 제공.

## ⚠️ 주의사항 (Disclaimer)
- **법적 결론**: AI 민원 가이드는 참고용이며 법률 자문이 아닙니다.
- **개인정보**: 민원 기록 시 학생/학부모의 실명 등 민감한 개인정보는 직접 입력을 피하시길 권장합니다. (AI는 입력된 개인정보를 반복하지 않도록 설정되어 있습니다.)
